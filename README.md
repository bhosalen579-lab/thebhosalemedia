# TBM Website

Architecture-only Next.js 15 scaffold with TypeScript, Tailwind CSS, App Router, Framer Motion, GSAP, React Three Fiber, Three.js, shadcn/ui conventions, and SEO metadata utilities.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
npm run type-check
```

## Structure

```text
src/
  app/                 App Router entry points and global styles
  components/          Shared components and shadcn/ui primitives
  config/              Site-level constants
  features/            Future feature modules
  hooks/               Shared React hooks
  lib/                 Utilities, SEO helpers, animation helpers
  providers/           App-wide providers
  styles/              Additional style modules if needed
  types/               Shared TypeScript types
```
